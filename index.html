📚 Introdução ao HTML

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Foi criado por Tim Berners-Lee em 1991.

Importância do HTML

  • Estrutura Básica: Todo site começa com HTML
  • SEO: Motores de busca usam HTML para indexar conteúdo
  • Acessibilidade: Leitores de tela dependem de HTML bem estruturado
  • Base para outras tecnologias: CSS e JavaScript complementam o HTML

Tags HTML Mais Comuns

Tag Descrição Exemplo
<html> Documento HTML Raiz do documento
<head> Metadados Título, links CSS
<body> Conteúdo visível Textos, imagens
<h1> a <h6> Títulos <h1>Título</h1>
<p> Parágrafo <p>Texto</p>
<a> Links <a href="#">Link</a>

Estrutura Básica de uma Página HTML

<!DOCTYPE html>
<html lang="pt">
    <head>
        <meta charset="UTF-8">
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Olá, Mundo!</h1>
        <p>Bem-vindo ao HTML.</p>
    </body>
</html>

🎨 Introdução ao CSS

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilo usada para descrever a apresentação de documentos HTML.

Tipos de CSS

  • Inline: Direto no elemento HTML
  • Interno: Na tag <style>
  • Externo: Arquivo .css separado

Vantagens do CSS

  • Separação entre conteúdo e estilo
  • Manutenção mais fácil
  • Carregamento mais rápido
  • Design responsivo

📦 Box Model

Margin - Padding - Border

Este elemento tem:

  • Margin: 20px (espaço externo)
  • Border: 3px sólido roxo
  • Padding: 20px (espaço interno)

Margin

Espaçamento externo entre elementos

margin: 10px 20px 30px 40px;

Padding

Espaçamento interno entre conteúdo e borda

padding: 15px;

Border

Borda ao redor do elemento

border: 2px solid black;
